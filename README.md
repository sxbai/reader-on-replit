# 使用说明

一键搭建脚本 (环境选择`Bash` 或 `Blank Repl`)   

Reader(单用户)一键脚本
```
bash <(curl -s https://raw.githubusercontent.com/sxbai/reader-on-replit/master/install.sh)
```

Reader(多用户)一键脚本
```
bash <(curl -s https://raw.githubusercontent.com/sxbai/reader-on-replit/master/install2.sh)
```
